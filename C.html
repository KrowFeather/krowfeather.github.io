4.1 关系表达式

关系运算符

关系运算符 含义
< 小于
> 大于
<= 小于等于
>= 大于等于
== 相等
!= 不相等
  • 关系运算符可用于每一种C语言数据类型。
  • 字符可以进行比较，会先转化为ASCII码再进行大小比较，如 'a'<'b'即为97<98 结果为真。

if语句

语法：

if(条件式)
     条件为真时执行的语句;
  • 条件式可以是任何有效的C语言表达式。(只不过最常见为关系表达式)
  • 在C语言中，任何非零值作为条件式的值，都会被认为是真，只有0为假。
  • 关系表达式只能产生两个数值之一，即0(假)和1(真)。
  • 关系表达式的每一个操作数可以是变量，常量或者任何有效的C语言表达式，但关系运算符只能为上表六个。

逻辑运算符

逻辑运算符 含义 用法
&&(二元) 与 当左右两个表达式都为真时，结果为真，否则为假
||(二元) 或 左右两个表达式只要其中任意一个为真则结果为真，否则为假
!(一元) 非 改变表达式的真假
  • 口诀：&&有假则假，||有真则真，！真假互换
  • &&的优先级比||高，结合性都是从左至右
  • &&和||都遵循短路求值特性(比如||的左边已经确定为真，则右边不会再进行计算，直接返回真)

优先级

从上到下优先级降低，同行运算符优先级相同

运算符 结合性
!,一元-,++,-- 从右到左
* / % 从左到右
+,- 从左到右
<,<=,>=,> 从左到右
==,!= 从左到右
&& 从左到右
|| 从左到右
+=,-=,*=，/= 从右到左

rand()函数与srand函数

用法：

#include<stdio.h>
#include<stdlib.h>   //rand()和srand()函数的定义在这个头文件中
#include<time.h>  //time()函数定义在此
int main(){
   srand(time(NULL));   //生成种子值
   int a=1+(int)rand()%100;   //产生随机数
   return 0;
}
  • 如果不使用srand()函数生成种子，直接使用rand()会进入伪随机状态，即只随机一次，之后再运行值不会再次随机。
  • rand()的随机值从0开始，即rand()%100实际上会返回一个0 ~ 99的随机数，1+rand()%100才为1 ~ 100。

第七章：函数模块性(2)

7.1 变量作用域

  • 在函数内部创建的变量即为此函数的局部变量，只能在该函数内使用
  • 全局变量即为在所有函数外定义的变量，能被所有的函数使用
//example
#include<stdio.h>
int a=1;//全局变量
void ex();
int main(){
   int b=1;//局部变量，只能在main函数中使用
   return 0;
}
void ex(){
   int b=1;//局部变量，只能在ex函数中使用
   b=2;//改变的是ex中的b，main中的b不受影响
}

存储类 用法

register 寄存器类，把变量存储在cpu里
static 静态存储类，一经创建，就会在生命周期中一直存在，只会初始化一次，不会被重新初始化

7.3 按引用传递

  • C语言分为按值传递和按引用传递(地址传递)两种传参方法
  • 按值传递不会修改实参的值，按引用传递会修改实参的值

取址符 '&'

取址符：&
通过加在变量的前面表示变量所在的内存地址

//example
#include<stdio.h>
int main(){
   int a;
   scanf("%d",&a); //这里用的就是获取a的地址
   return 0;
}

声明并使用指针

  • 能够存储地址的变量，被称为指针变量

定义指针变量方法: 数据类型 *指针名;

#include<stdio.h>
int main(){
   int b;

   int *p;  //声明指针变量p
   p=&b;    //p指向b的地址

   int *p1=&b; //也可以整合为一句
   return 0;
}
  • 声明一个指针变量后如果不同时为其赋初值(定义)(eg:char *str="hello"; )或不指向任何的地址(即没有p=&b)，则之后不能直接对其进行赋值
  • 指针变量的类型一定要与指向变量相一致(即int对应int* double对应double*)

第九章：字符串

9.1 字符串基础

字符串概念

  • 字符串字面值(字符串常量)是用双引号包含的字符序列 "Hello World" "This is a string"等都是字符串
  • 在C语言中，字符串保存为一个以NULL符号(值为\0)结尾的字符数组
  • \0代表着字符串的结束，不会被打印出来但仍占据一个数组长度
  • 双引号不是字符串的一部分
char name[10]="Ming" //占数组长度为5
char name[10]={'M','i','n','g'}; //占数组长度4

字符串输入和输出

函数 作用
gets() 输入字符串
puts() 输出字符串(自动换行)
getchar() 输入一个字符
putchar() 输出一个字符
  • gets()与scanf("%s",pt)的区别：gets只以换行符为结束字符串输入的标志，也就是说能够存储空格，而scanf以换行符和空格为结束输入的标志，也就是说空格以及后面输入的字符串都不会被存储进数组。
char string[20];
gets(name);
scanf("%s",name);
//输入 Hello World 
//若使用gets再进行输出则结果为 Hello World
//使用scanf则会输出 Hello
  • 由于数组名相当于保存这个数组的第一个内存位置的地址的指针常量，所以使用scanf+数组名输入字符串时，数组名前不能加'&'，但是若对字符数组的某单个位置进行赋值，则需要加'&'
char message[100];
scanf("%s",message);
scanf("%c",&message[0]);
  • scanf("%s",message);与scanf("%s",&message[0]);等价

第十章：数据文件

10.1 声明，打开和关闭文件流

  • 两种基本文件类型：文本文件和二进制文件
  • 文件流的相关函数和结构都在stdio.h头文件中

声明文件流

语法： FILE *fp(指针名);

#include<stdio.h>
int main(){
   FILE *fp;
   return 0;
}
  • 文件流的名称实际上是一个指针，所以需要*
  • FILE不是基本数据类型，而是一种数据结构

打开文件流

语法：指针名 = fopen("外部文件路径"，"指示符");

#include<stdio.h>
int main(){
   FILE *fp;
   fp=fopen("prices.txt","w");
   return 0;
}

打开文件流的目的：

  • 建立程序与数据文件之间物理通信的连接
  • 使声明的FILE指针名(内部名)等同于特殊外部文件名

文件未成功打开返回NULL值

指示符

指示符 描述
r 只读打开一个现存的文本文件
w 只写创建一个文本文件，如果文件存在，内容将被覆盖(丢失)
a 只写打开一个文本文件，新加入的文本会被追加到现存文件的末尾，若文件不存在则会创建一个新的文件
r+ 读写打开一个现存的文本文件，如果文件存在，则内容擦除
w+ 读写创建一个文本文件
a+ 读写打开文本文件，新文本追加在现存文件的末尾，若文件不存在，读写创建一个新文件
  • 记忆方法：r -> read(只读) w->write(只写) a->append(追加)
  • '+'实际上是对原模式附加没有的属性，即读与写属性全有，其他的属性与三个基本属性一致
  • 虽然r+具有读写属性，但注意仍然须为现存文件，若文件不存在不会创建，而是会和r一样返回NULL值

除此之外还有两种文件打开方式

指示符 描述
b 以二进制文件方式打开文件
t 以文本文件方式打开文件(不写默认为t)
  • '+'和b/t位置可以互换，即可以w+b也可以wb+

exit函数

函数包含在stdlib.h头文件中
功能：将整形参数直接传递给操作系统，然后终止程序
在主函数中exit(整型参数);等同于return (整数);
但在其他函数中二者有区别，return表示返回一个值而不结束整个程序，而exit仍然终止整个程序的运行

关闭文件流

语法：fclose(文件流指针名);

#include<stdio.h>
int main(){
   FILE *fp;
   fp=fopen("hello.txt","r");
   fclose(fp);
   return 0;
}

fclose()函数功能

  • 中断外部名与内部名之间的连接
  • 使内部名可以连接另外的文件

虽然在程序结束后操作系统会自动关闭打开的文件，但是使用fclose具有良好的意义(好习惯！)，因为所有计算机每次能够打开的最大文件数有上限(MinGW&MSVC为20)，当超过这个数再打开新文件即使存在也会返回NULL
